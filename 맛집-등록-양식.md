# 한입지도 맛집 등록 양식

아래 내용을 복사해서 메모에 붙여넣고 아는 항목만 작성하세요. 모르는 내용은 비워두거나
`모름`이라고 적어도 됩니다.

```text
[기본 정보]
맛집 이름:
지역 또는 주소:
분류: 한식 / 일식 / 중식 / 카페-디저트 / 기타

[방문 기록]
방문 여부: 방문함 / 가고 싶음
내 평점: 5점 만점
재방문 의사: 있음 / 없음 / 아직 모름
가격대:
맛집 전체 메모:

[나의 추천]
가장 추천하는 메뉴 1개:

[메뉴]
1. 메뉴 이름:
   가격:
   먹어본 여부: 먹어봄 / 안 먹어봄
   메뉴 메모:

2. 메뉴 이름:
   가격:
   먹어본 여부: 먹어봄 / 안 먹어봄
   메뉴 메모:

3. 메뉴 이름:
   가격:
   먹어본 여부: 먹어봄 / 안 먹어봄
   메뉴 메모:
```

메뉴가 더 많으면 같은 형식으로 번호만 늘리면 됩니다.

## 작성 예시

```text
[기본 정보]
맛집 이름: 홍콩반점 성수점
지역 또는 주소: 서울 성동구 성수동
분류: 중식

[방문 기록]
방문 여부: 방문함
내 평점: 4.2
재방문 의사: 있음
가격대: 1만원 이하
맛집 전체 메모: 혼자 간단하게 먹기 좋았음

[나의 추천]
가장 추천하는 메뉴 1개: 짬뽕

[메뉴]
1. 메뉴 이름: 짬뽕
   가격: 6,000원
   먹어본 여부: 먹어봄
   메뉴 메모: 단순하지만 국물이 맛있는 짬뽕

2. 메뉴 이름: 짜장면
   가격: 5,000원
   먹어본 여부: 먹어봄
   메뉴 메모: 양파가 가득한 진한 짜장면

3. 메뉴 이름: 양장피
   가격: 모름
   먹어본 여부: 안 먹어봄
   메뉴 메모:
```

## Codex에 요청하는 방법

작성한 내용을 채팅에 붙여넣고 다음처럼 요청하면 됩니다.

```text
이 맛집을 한입지도에 추가하고 테스트한 뒤 GitHub에 올려줘.
```
